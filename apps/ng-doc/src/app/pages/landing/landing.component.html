How it works

Markdown of NgDoc supports Nunjucks, so you can extend, include and reuse your markdown templates which making them truthfully dynamic! NgDoc includes two different types of demos, so you can choose the one that suits you best and display it via just one line of code! NgDoc provides playgrounds that read inputs of your components and directives and creates a live example of it! NgDoc can automatically generate and update API Documentation based on your code! NgDoc automatically indexes everything you write to create a better search experience for your users You can use keywords to link your guides and API, NgDoc will automatically update them! Keywords also work in the code blocks! Interface is fully customizable! You can change colors, fonts, sizes, replace sidebar or header with your own components and much more!
